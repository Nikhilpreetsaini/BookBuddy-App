📚 Welcome to BookBuddy!

This is a minimal front‑end for the BookBuddy API. The real power of BookBuddy lies in its REST endpoints – use your favourite API client (such as Postman) to register, log in, add books, record reading sessions and interact with other readers. You can start exploring the API at / which returns a welcome message.

For more information check out the GitHub repository and the README there.
